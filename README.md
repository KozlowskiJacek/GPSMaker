# GPSMaker – User Guide

GPSMaker is a simple script for processing GPS reports in CSV format.  
It allows you to reduce the time required to process reports from several days to just two clicks.

---

## Requirements

- Python (version 3.x)  
- **pandas** library

To install Pandas, run the following command in your terminal or command prompt:

```bash
pip install pandas
How to Use
1. Prepare the report file
Download the GPS report in CSV format.

Place it in the input folder located in the GPSMaker directory.

2. Run the script
Double-click the GPSMaker.py file.

Follow the instructions displayed in the console.

3. Get the result
The processed, "clean" CSV/Excel file will be saved in the output folder.

The file is ready for further use.

Advantages
Automatic processing of GPS reports.

Saves time – a process that used to take 3–4 days now takes only a few seconds.

Notes
Make sure that only CSV files to be processed are placed in the input folder.

The output folder will be created automatically by the script if it does not exist.
